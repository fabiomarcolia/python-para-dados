## Como deixar o GitHub “premium”

- Adicione prints em `projects/<modulo>/assets/` e referencie no README do mini-projeto.

- Use o **projeto final (99)** como peça principal e os mini-projetos como evidência incremental.

- Recomendações: fixe o repositório no perfil e adicione uma imagem de Social Preview.
