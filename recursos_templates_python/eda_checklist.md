# Checklist de EDA (para copiar e colar)

1- Contexto
- Qual a pergunta de negócio?
- Quais métricas importam?

2- Qualidade
- Tipos (dtype), nulos, duplicados
- Valores fora do domínio (ex.: idade negativa)

3- Distribuições
- Numéricas: hist/boxplot, outliers
- Categóricas: frequência, top N

4- Relações
- Correlação (com cuidado), scatter, pairwise
- Segmentações por categoria/tempo

5- Tempo (se houver)
- tendência, sazonalidade, janelas

6- Conclusões
- 3–5 insights com evidências (gráfico + texto)
- Riscos/limitações (viés, amostra, missingness)

7- Próximos passos
- features, modelo baseline, coleta de dados, etc.
