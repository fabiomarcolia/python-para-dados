# BRIEF — 00_setup

## Objetivo
Padronizar seu ambiente e garantir que qualquer pessoa consiga rodar o repositório no VS Code.

## Tarefas
1- Criar o ambiente: `.venv` + `requirements.txt`
2- Rodar pelo menos 2 scripts do repo via terminal
3- Rodar pelo menos 1 notebook via VS Code (Jupyter)
4- Configurar `Python: Select Interpreter` apontando para `.venv`
5- (Opcional) adicionar um badge no README: Python version / Jupyter / etc

## Evidências (portfólio)
- Print do VS Code com kernel ativo
- Print do comando executando scripts
- Um parágrafo no README do projeto: “Como rodar”

## Checklist rápido
- `pip install -r requirements.txt` ok
- `python -c "import pandas; import numpy"` ok
- Notebook abre sem erro
