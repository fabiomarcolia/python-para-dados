# Mini-projeto — 03_pandas (EDA Básica)

## Objetivo
Escolher 1 dataset do repositório de bases e entregar uma **EDA limpa e bem explicada**.

## Entregáveis (portfólio)
- `notebook.ipynb` (EDA)
- `README.md` (contexto + insights)
- `assets/` (gráficos exportados)

## Sugestão de roteiro
1- Entendimento do dataset (colunas, tipos, granularidade)
2- Qualidade (nulos/duplicados/tipos)
3- Perguntas de negócio (3 a 5)
4- Análises (groupby, agregações, segmentações)
5- Conclusões (insights + próximos passos)
