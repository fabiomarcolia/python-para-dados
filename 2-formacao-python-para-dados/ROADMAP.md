# Roadmap (linear)

> Aulas pensadas para 15–30 min cada, com prática e mini-projeto por módulo.

## 00_setup
- Ambiente (venv), VS Code, Jupyter
- Organização do repositório e rotina de estudo

## 01_fundamentos (revisão focada em dados)
- estruturas, funções, comprehensions, arquivos
- erros/exceções, context manager, módulos

## 02_numpy
- arrays, broadcasting, boolean indexing
- operações vetorizadas (base para performance)

## 03_pandas
- ingestão, limpeza, joins, groupby, pivot
- datas, texto, categorias, boas práticas de EDA

## 04_visualizacao
- matplotlib (base), seaborn (exploração), plotly (interativo)
- escolha do gráfico + narrativa

## 05_polars
- expressions, lazy vs eager
- scan de arquivos grandes, performance e pipeline

## 06_pyarrow_parquet
- parquet, schemas, particionamento
- interoperabilidade pandas/polars

## 07_eda_avancado
- missingness, outliers, validações
- features úteis, relatório reprodutível

## 08_machine_learning
- pipelines, split, métricas
- baseline, cross-validation, tuning leve
